After downloading and unzipping the file, paste the data to Google Sheets.

Min, Max

First, use the MIN and MAX functions to check the minimum and maximum values of numeric fields such as impressions, spent, clicks, leads, orders, and revenue. It can also be used to check the date range.

Copy the headers to a new sheet and use rows 2 and 3 to show the minimum and maximum values respectively

Use =MIN(Sheet1!B:B) to find the minumum (earliest) date in column B of Sheet1. For the maximum date (cell B3), simply replace MIN with MAX: =MAX(Sheet1!B:B)

The same logic applies to other numeric columns — just change the target column accordingly.

In terms of formatting, set the date column to "Date" via Format → Number → Date from the toolbar on the top. Use standard number format with comma separators for better readability.

Then we can see the data is about digital marketing performance in February 2021. Metrics ranges are wide, possibly due to results from different campaign types.

Sumif, Countif

To further explore the dataset using campaign_name and category, we first need to organize data.

campaign_name field includes platform names — Facebook, Instagram, Google, YouTube, etc. And that can be extracted for further analysis.

The structure of campaign name looks like platform + underscore + campaign type. Use FIND and LEFT to identify the underscore and extract the platform name:

=LEFT(C2, FIND("_", C2) - 1)

Use LOWER to make all text lowercase (standardize):

=LOWER(LEFT(C2, FIND("_", C2) - 1))

Or use a seperate column for clarity:

=LOWER(L2)

Now we can aggregate data by category and platform.

In a new sheet, get an unique list of categories by setting:

=UNIQUE(Sheet1!D:D)

Use SUMIF to get totals. Example (impressions for "social"):

=SUMIF(Sheet1!$D:$D, $A2, Sheet1!F:F)

Remember to put dollor sign $ to lock column/row so that it's easier to drag formula across the sheet.

Summ one metric for one category and drag the formula across rows and columns to get the full data.

Then calculate key metrics:

  • CTR = clicks / impressions
  • CPC = spend / clicks
  • CPL = spend / leads
  • CPA = spend / orders
  • ROAS = revenue / spend
  • Average order price

Compute percentage of clicks, leads, orders, and revenue per category.

Observations:

  • Social and Media bring ~90% of impressions
  • Leads and orders mostly from Social and Influencer
  • Influencer channel yields highest ROAS ($2.54); media is second ($1.22)

The results overall align with general marketing performance: display/media ads drive exposure while conversions and revenue come from other channels. Consider increasing spend on high-performing channels like Influencer. Monitor performance closely since the margin may go down as spending goes up.

To dive deeper, analyze platform, campaign, and daily changes. That's covered in the next section.

Currently, we have four sheets. Give them meaningful names like raw, maxmin, category, and platform.

All of the above can also be done using a pivot table.

Pivot Table

In the raw sheet, select the full data range. From the menu, choose Insert → Pivot Table.

Use the Pivot Table Editor to build your view.

Google Sheets supports automatic grouping by category or platform.

Clicks %: Add clicks twice under Values, then set one to "% of grand total".

CPC: Add a Calculated Field:

=SUM(mark_spent)/SUM(clicks)

Rename the field to "CPC".

Setting up a pivot table takes more time initially, but once it's ready, it can be easy to explore different dimensions (category, platform, date, or campaign) and make different versions quickly.

INDEX + MATCH

INDEX with MATCH is a powerful combo. While it may seem complex at first, but it's really flexible once you get the hang of it.

The first argument of INDEX is the column from which you want to retrieve data — in this case, the category column (column D) of the raw data. The second argument is the row number. In this example we put 2 and get the value in cell D2, which is "social". Since this row number will be dynamic, we use MATCH to find it.

MATCH needs three arguments:

  • Lookup value: e.g., values in column A of the platform sheet (facebook, google, youtube, etc.)
  • Lookup range: where to search for the value — in this case, column L of the raw sheet (platform)
  • Match type: usually 0, which means exact match

For example, to find the value of A3 in the platform sheet (which is "facebook"), MATCH searches column L of the raw data and returns the row number of its first appearance, which is 2. This value is then used as the row number in the INDEX function.

There are two other functions related to referencing values by position: VLOOKUP and INDIRECT. VLOOKUP is more commonly used in practice than INDIRECT. These two are not the best fits for this specific case, though.

The lookup exercises below are mainly for practice, as there's no strict one-to-one relationship between platform and category. A single platform might appear under both "social" and "influencer". In such cases, the first match is returned.

VLOOKUP

VLOOKUP searches for a value in the first column of a data range and returns a value from a specified column in the same row.

It takes three required arguments:

  1. Lookup value
  2. Table range
  3. Column number (to return from)

It also has an optional fourth argument for match types: exact (0) or approximate (1). Approximate (1) is the default.

For example, in a blank cell of the raw sheet, set:

=VLOOKUP("social", D1:L8, 2, 0)

This searches column D in the range D1:L8 for the first occurrence of "social" and returns the value in the second column of that row. In this case it's the impressions column.

VLOOKUP can only search from left to right. Re-arrange the data before using VLOOKUP if the lookup value isn't in the first column of the specified range,

Although the fourth argument is optional, it's better to match the value precisely. Set it to 0 for exact match. (Default value in both Excel and Google Sheets is approximate)

More on official docs:

  • Excel VLOOKUP
  • Google Sheets LOOKUP

INDIRECT

INDIRECT takes one required argument, a string representing a cell reference. Like "D2" or "Sheet1!A2".

It's often used when consolidating data from multiple sheets to a summary sheet. For example, if monthly data is stored in separate sheets (January, February, etc.) and you want to grab them all in a summary sheet, INDIRECT would be very handy.

Chart Setup

Freeze top rows so that they stay in place when dragging formulas down.

Add a "Day of Week (DOW)" column to show Mondays, Tuesdays, etc., so that it's easier to see the difference in performance between weekdays and weekends.

Get numbers that represent weekdays from the WEEKDAY function. By default, Sunday is 1 and Saturday is 7, but this can be customized using the second argument.

WEEKDAY function documentation

Then, pass results to the TEXT function to get text values like "Monday" or "Mon". TEXT takes two arguments: target value and format. Common formats for dates are "dddd" (e.g., Monday) and "ddd" (e.g., Mon).

TEXT function documentation

We can see that there is no significant difference between weekdays (Mon-Fri) and weekends (Sat-Sun) performance. Instead, Fridays and Saturdays account for the most orders.

Let's visualize daily performance trends using Combo Charts. We'll create three charts:

  • Clicks (bar) + CPC (line)
  • Impressions (bar) + CTR (line)
  • Marketing Spend (bar) + ROAS (line)

Start with Clicks + CPC. Go to Insert → Chart, select Combo Chart in the Chart Editor. Set the Data Range to include c_date (A), clicks (E), and CPC (G).

Add c_date to X-axis, and both clicks and CPC to Series. Then go to Customize → Series and set CPC to the right axis.

Observations:

Clicks and spending peaked around mid-month (Feb 10–20). On Feb 18, impressions spiked but spend decreased. This could be due to ad budget being focused around Lunar New Year (Feb 12–16, 2021).

Using a pivot table to check traffic sources, we find the ad campaign behind the Feb 18 spike in impressions was banner_partner.

Other metrics like clicks, spend, and orders did not increase significantly on the same day. It's possible that some camapigns are charged based on orders and not impressions. It's also noticed that order volumn peaked after impressions spiked (about 2 days after).

Sorting Issues with Formula Columns

Columns generated by formulas cannot be sorted directly. There are two ways to revolve it:

  • Named Ranges
  • Filter (views or functions)

Named Ranges

Select campaign data (excluding header) and name the range campaign_performance.

In a new sheet, enter the following formula in cell A2:

=sort(campaign_performance,3,FALSE,2,TRUE,6,FALSE)
SORT function documentation

Observations:

  • Highest average order value: influencer
  • Media provided the most impressions
  • Top performers (increase budget if possible): instagram_blogger, youtube_blogger
